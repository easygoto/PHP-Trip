## 项目结构
```
app : 项目目录
  |--Demo : 暂时无法分类的东西
  |--Logic : 逻辑代码
config : 配置文件
  |--不解释
dp : 学习设计模式
  |--各种设计模式
res : 资源文件
  |--不解释
src : 组件、库、工具等
  |--Component : 组件
  |--Helper : 工具
  |--Library : 库
tests : 测试板块
  |--App
  |--Core
  |--Dp
```
