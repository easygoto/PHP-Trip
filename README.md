## 项目结构
```
app : 应用目录
config : 配置文件
  |--不解释
dp : 设计模式
  |--各种设计模式
res : 资源文件
  |--不解释
src : 组件、工具等
  |--Component : 组件
  |--Container : 容器
  |--Helper : 工具
  |--Library : 库，废弃，建议使用工具
tests : 测试板块
  |--App
  |--Core
  |--Dp
```
